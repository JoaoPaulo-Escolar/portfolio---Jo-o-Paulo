# 👨‍💻 Portfólio de João Paulo

> Um portfólio moderno e responsivo desenvolvido em HTML, CSS e JavaScript puros.

## 📋 Sobre

Portfólio apresentando minha trajetória como estudante de desenvolvimento de sistemas, com informações sobre formação acadêmica, habilidades e formas de contato.

## 🎯 Seções

- **Pessoal**: Apresentação, sobre mim, habilidades e idiomas
- **Acadêmico**: Informações sobre os 3 anos de ensino médio técnico
- **Redes Sociais**: Links para redes sociais e formas de contato

## 🎨 Características

- ✨ Design moderno e minimalista
- 📱 Totalmente responsivo
- 🎭 Tema escuro profissional
- 🎵 Player de áudio integrado
- 🎬 Suporte a vídeos
- ⚡ Animações suaves
- 🔗 Navegação intuitiva

## 🛠️ Tecnologias

- HTML5
- CSS3 (com variáveis CSS)
- Fonts do Google (Space Mono, Syne)

## 🚀 Como usar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/portfolio.git
```

2. Abra o arquivo `index.html` em seu navegador

## 💻 Personalização

### Cores
Você pode customizar as cores editando as variáveis CSS em `css/style.css`:

```css
:root {
  --red: #ff3333;
  --bg: #0a0a0f;
  --text: #e8e8f0;
  /* ... mais variáveis */
}
```

### Conteúdo
- Atualize os links de redes sociais em `pages/sociais.html`
- Modifique informações pessoais em `index.html`
- Adicione suas próprias imagens e mídia em `assets/`
